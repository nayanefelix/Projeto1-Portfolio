Exemplo de página 1

Esse é um exemplo de página web minimalista, ideal para textos. Aqui você pode colocar o conteúdo que quiser. Essa página foi construída utilizando HTML e CSS. No corpo da página, defini a cor do fundo utilizando o comando background-color na tag body do código. O título e o subtítulo foram separados numa div class e marcados utilizando as tags h2 e h4. Seus aspectos estéticos como posição, fonte e cor foram definidos no CSS utilizando os comandos text-align, font-family e color, dentro da div class criada. Ainda dentro dessa classe, utilizei os comandos margin, padding e border para definir as margens e o estilo das bordas.
