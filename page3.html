Exemplo de página 2 | Flexbox

Esse tipo de layout é ideal para um portfólio de fotos. A página foi feita no HTML/CSS, em formato de Flexbox, utilizando a div id container. Dentro desse container, foram inseridas divs e classes para colocar os elementos dentro da Flexbox. Defini o background no CSS utilizando o comando background: url ( ). Utilizei também os comandos display: flex, flex-direction, align-items, justify-content e margin para posicionar tanto a Flexbox quanto os itens dentro dela.

Div é, por natureza, block, enquanto span é in-line. Isso significa que a tag div automaticamente dispõe os itens em blocos, e a tag span dispõe em linhas.

Display: none esconde os elementos, display: inline aplica os comandos no elemento, mas não muda o espaço externo, display: block bloqueia toda a linha, display: inline-block dispõe em linhas e aceita mudanças de tamanho, e o display: flex coloca os elementos num flex-container.

É de acordo com a flex-direction que o navegador organiza os elementos colocados dentro do flex-container. Ela pode ser row, row-reverse, column ou column-reverse.
